# `.claude/commands/`

Slash commands for Claude Code sessions. Each `.md` file here becomes a `/command`
that agents can invoke.

**What belongs here:** reusable prompts and workflows — e.g. `/new-package`,
`/review`, `/explain`.

**Owning ticket:** D2 — the first commands (including `/new-package`) will be
authored there. This directory is scaffolded by D1 as a placeholder.
